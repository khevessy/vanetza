  • API Documentation
  • Edit on GitHub

An API documentation can be generated directly from source code with Doxygen. If Doxygen is installed on your system, you may simply call doxygen in the repository’s root directory. The generated HTML documentation can then be found at doxygen/html/index.html.

We also maintain an online copy which is updated every now and then. Of course, your locally generated documentation will be more up-to-date in most cases. Please consider to generate the documentation locally if you need it to match a particular or more recent version of Vanetza.
